# 📚 Book Tracker (Трекер Прочитанных Книг)

Простое и удобное консольное приложение на Python для отслеживания ваших прочитанных книг, оценки и статистики по авторам.

## ✨ Основные функции

*   **Добавление книг:** Ввод названия, автора, оценки и даты прочтения.
*   **Просмотр библиотеки:** Отображение всех добавленных книг в аккуратном виде.
*   **Статистика:** Расчет общей средней оценки по всем книгам.
*   **Аналитика:** Анализ по авторам — отображение средней оценки для каждого автора.
*   **Удаление:** Возможность удалить книгу из базы данных по номеру.
*   **Автосохранение:** Все данные сохраняются в файл `books.json` в той же директории, что и скрипт.

## 🚀 Как запустить проект

### 🛠️ Требования
*   Python 3.x

### 🐍 Установка зависимостей
В данном проекте используются только стандартные библиотеки Python (`json`, `os`, `datetime`). Ничего устанавливать не нужно!

### ▶️ Запуск
1.  Убедитесь, что вы находитесь в корневой папке проекта.
2.  Выполните команду:
    ```bash
    python book_tracker.py
    ```

## 📚 Использование (Меню)

При запуске вы увидите интерактивное меню:

1.  **Добавить книгу:** Позволяет ввести все детали книги.
    *   *Примечание:* Если вы введете некорректную дату, система попросит вас ввести дату снова, пока не будет получен корректный формат `ГГГГ-ММ-ДД`.
2.  **Показать все книги:** Выводит список всех записей.
3.  **Показать среднюю оценку:** Вычисляет общий средний балл.
4.  **Статистика по авторам:** Группирует книги по авторам и рассчитывает средний балл для каждого из них.
5.  **Удалить книгу:** Показывает список и позволяет удалить запись по номеру.
6.  **Выход:** Сохраняет данные и завершает работу программы.
